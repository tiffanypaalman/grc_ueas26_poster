# Pits &amp; Pieces: Investigating the relative effects of habitat degradation and food web alteration on ecosystem recovery, structure, and function

This is the GitHub repository for the research project investigating the relative effects of degree of degradation and ray foraging disturbance on oyster reef recovery, structure, and function, presented at the GRC: Unifying Ecology Across Scales from July 11th - 17th, 2026.
